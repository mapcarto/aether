# 多维空间网格与体素化计算 (Spatial Grid & Voxelization)

> 旨在将复杂的空间拓扑学原理、离散数学结合极致的零拷贝 C 源指针。

## 1. 原理与局限：为什么抛弃传统空间索引？
传统的树状结构 (四叉树 Quad-tree、包围盒 R 树) 及静态离散体系 (S2、Uber H3) 将对象视为不可分割的外包盒，这对于存在大空洞与极端长条形的复杂实体会导致巨大的“查询假阳性 (False Positives)”。

## 2. 自动层级选择：无瑕疵的 LOD 映射
1. **逆向回溯算法**：从高精度的最低细粒度网格向上遍历。
2. **停止条件**：网格跨度显著数量减少即判定最佳匹配层。
3. **结果**：对象的物理尺寸与其在金字塔中的节点大小达成完美契合，原生支持无限 LOD 适配。

## 3. 金字塔内部的 ECS 无缝融合
- 网格单元并非游离架构，它就是一个特殊的 **ECS 基本实体**。
- **主卡与子卡片生命周期**：共享状态独立维护，头指针通过位掩码被组件直接关联访问，全由统一 ECS 生命周期驱动。

## 4. 体素化离散与 4D 时空穿梭
从莫顿码 (Morton Code) 生成，至行进立方体 (Marching Cubes) 抽取的流程；及四维时间维度检索的能力集成。

## 5. 分区回调 (Partition Callback)
> 极粗糙的躯干留在低层级，碰撞精度需求极高的持枪指尖沉入高精度层级。

- 在对象初入索引网络时即注册的拆解函数。
- 最终结果**写入底层 GPU 连续缓冲区与内存中块**。
- 零二次搬移 (Zero-copy Data Transfer) 以释放系统总线带宽极限。
