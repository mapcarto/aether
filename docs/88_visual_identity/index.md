# 视觉识别系统 (Visual Identity & Assets)

此文档域（Domain）用于集中存放与维护 Aether 引擎品牌体系、外围 UI/UX 组件以及宣传物料的设计规范。

基于 Aether “底层剥离”的极客调性，视觉体系同样旨在传达：**极致的网格化、工程纪律、纯净无锁**。
